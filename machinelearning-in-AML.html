INTRODUCTION

Financial Institutions around the world spend millions if not billions of dollars on hiring resources or outsourcing the work of manual investigation of the alerts generated by the traditional rule based system. as to comply with Anti Money Laundering Regulations imposed by the respective regulators.


Global financial services companies will pay $180.9 billion in compliance costs for year 2020, according to a new report from LexisNexis Risk Solutions.

Read more

What is the PROBLEM!?

Currently many of the organizations are using a tradional rule based alert system which is essentiallly just a set of rules and if the account satisfies those rules then an alert will be generated and the account will be sent to a human FinCrime Analyst for investigation. A human investgator spends an hour to investiagte the case on an average invbolving many repetative tasks. A huge problem faced by the institutions is the number of false positive and accourding to some of the reports 75-90% of the cases which are investigated by human analyst upon a alert generated by rule based alert system.

What is the SOLUTION?

Okay so we now have an understanding of the problem and now lets discuss how we can try to solve the problem.


We are trying to devlop a supervised machine learning algorithm with an aim not to replace the conventional rule based syatem but to reduce the number of false positive that is we are trying to decrease the number of fasle positive cases by training the machine learning model by the dataset which is generated by properly solved cases by a human analyst so that we have good quality dataset to train our model. The idea is to take solved cases which was triggered by rule based alert system and as the traning data and not to directly impliment the ML algorithm on the system. The goal is not to eliminate but to make the existing system BETTER.

What will be the IMPACT!?

The fact that this algorithm is aimed to reduce number of false positive cases is the reason that institutions now a days are more inclined to upgrade their existing alert system and to save millions on dollars in operations and this algorithm will prove as a great tool to achieve this goal.
